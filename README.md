# Market Customer Segmentation

## Objective
Segment customers based on purchasing behavior using Machine Learning.

## Technologies
- Python
- Pandas
- Matplotlib
- Scikit-Learn

## Techniques
- Data Preprocessing
- RFM Analysis
- PCA
- K-Means Clustering
- Cluster Profiling

## Run Project

pip install -r requirements.txt

python main.py
